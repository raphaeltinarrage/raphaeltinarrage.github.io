Topological Data Analysis with Persistent Homology

(Potential program)

First week: Basics of topology
  • 26/01 (2pm-4pm) - Topological spaces
    Axiomatic definition of topological spaces, continuous maps, homeomorphic topological spaces.
  • 27/01 (2pm-4pm) - Homotopy equivalence
    Homotopy equivalent topological spaces, examples, discussion about topological (homeomorphic and homotopic) invariants.
  • 28/01 (2pm-4pm) - Simplicial complexes
    Definition of simplicial complexes, their topology, nerve theorem, Euler characteristic.
  • 29/01 (2pm-4pm) - Python tutorial
Second week: Homology
  • 01/02 (2pm-4pm) - Singular homology
    Definition of singular homology (over finite fields), Mayer-Vietoris sequence, examples (homology of spheres), application (invariance of domain, Brouwer's fixed point theorem).
  • 02/02 (2pm-4pm) - Simplicial homology
    Definition of singular homology, connection with singular homology, Betti numbers, algorithm.
  • 03/02 (2pm-4pm) - Topological inference
    Discussion about applied aspects of topological inference, problem of the scale, Cech filtration.
  • 04/02 (2pm-4pm) - Python tutorial
Third week: Persistent homology
  • 08/02 (2pm-4pm) - Persistence modules - Decomposition
    Formal definition of persistence modules, interval-modules, decomposition theorem of Crawley-Boevey, barcodes.
  • 09/02 (2pm-4pm) - Persistence modules - Stability
    Interleaving distance, bottleneck distance, isometry theorem of Lesnik, stability theorem of Cohen-Steiner - Edeslbrunner - Harer.
  • 10/02 (2pm-4pm) - Python tutorial
